
# Global Developer Salary Analysis and Outlier Detection

## Project Overview

This project performs Exploratory Data Analysis (EDA) on global developer salary survey data.  
The goal is to understand yearly compensation patterns, clean missing data, detect outliers, and visualize salary distributions across countries.

This project demonstrates core data analyst skills such as data cleaning, statistical analysis, and data visualization using Python.

---

## Objectives

- Analyze the distribution of yearly compensation  
- Remove missing salary and country values  
- Detect salary outliers using the IQR method  
- Visualize salary distributions using histograms and box plots  
- Compare salary trends across top countries  

---

## Tools and Libraries Used

- Python  
- Pandas  
- Matplotlib  
- Seaborn  

---

## Key Analysis Steps

- Load and inspect the dataset  
- Handle missing values  
- Visualize overall salary distribution  
- Calculate quartiles and IQR  
- Detect high-end salary outliers  
- Visualize outliers using box plots  
- Perform country-wise salary comparison  
  

---

## Key Findings

- Salary data is right-skewed  
- No lower-end salary outliers were found  
- High-end salary outliers exist above $220,860  
- Salary distributions vary across countries
